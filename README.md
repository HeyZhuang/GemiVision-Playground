# CH-Love.online - 爱的圣诞树 🎄

一个浪漫的3D圣诞树网站，展示美好的回忆照片，支持触摸和手势控制。

## 🌟 功能特色

- **3D圣诞树展示**：使用Three.js创建的华丽3D圣诞树
- **照片展示**：将珍贵回忆照片装饰在圣诞树上
- **多种交互模式**：
  - 🎄 圣诞树模式：照片围绕圣诞树排列
  - ✨ 星空散落模式：照片在3D空间中自由飘散
  - 🔍 随机聚焦模式：随机选择照片进行特写展示
- **手势控制**：支持MediaPipe手势识别
  - 张开手掌：旋转视角
  - 手掌前后移动：缩放视图
  - 双击握拳：切换日夜灯光模式
- **触摸控制**：支持移动设备触摸操作
- **全球访客统计**：实时显示访客分布和统计数据
- **响应式设计**：完美适配各种设备屏幕

## 🛠️ 技术栈

- **前端框架**: Three.js (3D渲染)
- **手势识别**: MediaPipe
- **后处理效果**: Three.js后处理管道
- **统计服务**: 不蒜子访客统计
- **地图服务**: MapMyVisitors全球访客分布
- **部署**: Nginx + 自动化CI/CD

## 📱 操作指南

### 桌面端操作
- **鼠标拖拽**：旋转视角
- **点击照片**：聚焦查看特定照片
- **按 'S' 键**：显示/隐藏访客统计面板
- **底部按钮**：切换不同展示模式

### 移动端操作
- **触摸拖拽**：旋转视角
- **点击照片**：聚焦查看
- **双击屏幕**：切换散落/圣诞树模式

### 手势控制（需摄像头权限）
- **张开手掌**：通过手部移动控制旋转
- **手掌前后移动**：控制缩放距离
- **双击握拳**：切换日间/夜间灯光模式

## 🔧 本地开发

1. 克隆项目：
```bash
git clone https://github.com/HeyZhuang/GemiVision-Playground.git
cd GemiVision-Playground
```

2. 直接在浏览器中打开 `index.html` 文件即可运行

3. 或使用本地服务器：
```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .
```

## 🚀 部署说明

### 自动部署
项目配置了GitHub Actions自动部署，每次推送到main分支时会自动部署到服务器。

### 手动部署
使用提供的部署脚本：
```bash
chmod +x deploy.sh
./deploy.sh
```

### 服务器配置
- **服务器**: Ubuntu Linux
- **Web服务器**: Nginx
- **域名**: ch-love.online
- **SSL**: 建议配置Let's Encrypt证书
